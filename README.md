# IoT Home Automation System (ESP32)

## Overview
Wi-Fi based home automation system using MQTT protocol.

## Features
- Remote appliance control
- Low latency (<200ms)
- MQTT communication

## Tech Stack
ESP32, Embedded C, Python, MQTT

## Results
- Achieved 99% uptime in stress testing
- Reliable real-time control

## Future Scope
- Mobile App Integration
- Voice Control (Alexa/Google Assistant)
